# アドベンチャーゲーム
日本工学院八王子専門学校 ゲーム科 実習用サンプル。

## What is this
HTML/CSS/JavaScriptの学習用の簡易的なアドベンチャーゲームです。

## 準備
### リポジトリの取得
```shellsession
$ git clone https://github.com/katsube/neec-adv-2023.git adv
```

### 実行方法
#### ローカル
adv内にある`index.html`をWebブラウザから参照してください。

#### Webサーバ
実習用サーバのhtdocs下に`adv`など適当な名前のディレクトリを作成しWinSCPなどでアップロードしてください。

Webブラウザからアクセスします。
> https://(小文字の学籍番号).neecbox.net/adv/
